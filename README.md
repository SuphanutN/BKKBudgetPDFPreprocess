# BKKBudgetPDFPreprocess
jupyter for pre-precess text that extract from PDF file (using https://document.online-convert.com/convert-to-txt) 

## Method

1. Download File from BKKDream Datasource (PDF File)
2. Use online-convert to transfrom PDF to text file
3. Use jupyter to preprocess text and extract topic/subtopic and save file as .csv file
4. explore using jupyter for review the overall data
